# Assets Folder

Place your company logos and other static assets here.

## Required Logo Files

Add the following logo files to this folder to display them in the Experience section:

### Experience Logos:
1. **DoMedia Company** - Save as `domedia-logo.png`
   - Download from: https://www.domedia.lk/

2. **United Motors Pvt Ltd** - Save as `united-motors-logo.png`
   - Download from: https://www.unitedmotors.lk/

### Education Logos:
3. **SLIIT** - Save as `sliit-logo.png`
   - Download from: https://www.sliit.lk/

4. **Open University Sri Lanka** - Save as `open-university-logo.png`
   - Download from: https://www.ou.ac.lk/

5. **Automobile Engineering Training Institute** - Save as `aeti-logo.png`
   - Download from their website or use a placeholder

## Supported Formats
- PNG (recommended for transparency)
- JPG
- SVG (scalable)

## File Naming Convention
Use lowercase with hyphens:
- `domedia-logo.png`
- `sliit-logo.png`
- `open-university-logo.png`

## Notes
- Logos will automatically appear next to each experience/education entry
- If a logo file is missing, it will be hidden gracefully without breaking the layout
- Recommended size: 400x400px or larger (will be displayed at 96x96px)
